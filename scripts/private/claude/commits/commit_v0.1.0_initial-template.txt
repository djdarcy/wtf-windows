v0.1.0: feat: initial project template with PyPI and ComfyUI archetypes

Template repository for DazzleTools Python projects. Provides
baseline project structure with $placeholder variables for
project-specific values.

Structure:
- .github/ (CI, issue templates, PR template, CODEOWNERS, funding,
  dependabot, stale bot, release template)
- tests/ (conftest, test_version, one-offs/ with graduation README)
- docs/, private/claude/ (directory structure placeholders)
- pyproject.toml.pypi (PyPI packaging with setuptools)
- pyproject.toml.comfyui (ComfyUI registry format)
- .gitignore (repokit standard with private-content protection)
- .repokit.json, CONTRIBUTING.md, LICENSE (GPL-3.0)

Placeholder variables: $PROJECT_NAME, $PACKAGE_NAME, $DESCRIPTION,
$GITHUB_ORG, $GITHUB_USER, $AUTHOR_EMAIL, $CLI_COMMAND
